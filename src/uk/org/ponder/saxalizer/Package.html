Tools for serializing and deserializing Java objects to and from the XML format.

The SAXalizer uses a standard SAX parser to read XML data, and deserializes it onto either user-defined objects implementing the SAXalizable interface, or else onto Generic DOM-like hold-all objects of type GenericSAXImpl.

The DeSAXalizer runs this process in reverse, and serializes either DeSAXalizable or GenericSAXImpl objects into an XML stream.
